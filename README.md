## Installation 

1. Clone the project
   ```sh
   git clone https://github.com/Sebastian-Paulluk/Curso-React-Paulluk.git
   ```
2. Navigate to the project directory
   ```sh
   cd curso-react-paulluk
   ```
3. Install dependencies
   ```sh
   npm install
   ```
4. Run the app
   ```sh
   npm run start
   ```
